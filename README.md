# Online Appendix

**Paper:** Technical Assumptions in SOX Section 302/404 and the Autonomous Agent Challenge: A Structured Assumption–Violation Mapping of Internal Controls Over Financial Reporting in Agentic AI Environments

**Submitted to:** Journal of Emerging Technologies in Accounting (JETA), Special Section: Agentic AI in Auditing and Accounting

---

## Contents

### Online Appendix A: COSO Principle-by-Principle Analysis

Complete mapping of all 17 COSO Internal Control Framework principles to autonomous AI agent behaviors. For each principle, the appendix identifies the human assumption embedded in the principle and the specific way in which autonomous agent properties violate that assumption. The main paper presents the summary in Table 3; this appendix provides the full analysis.

### Online Appendix B: Detailed COSO Extension Definitions

Complete definitions of all 17 proposed COSO extensions for agentic AI environments. Each extension specifies the operational requirements for organizations deploying AI agents in financial reporting processes. The main paper presents the summary in Table 6; this appendix provides the full extension specifications.
